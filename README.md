# 📚 Sistema de Gestão Escolar

Projeto desenvolvido como **tarefa acadêmica** para a faculdade, utilizando **Python** e **MySQL**.  
O objetivo foi implementar um sistema simples de **CRUD** (Create, Read, Update, Delete) para cadastro e gerenciamento de alunos e disciplinas em uma escola.

---

## 🎯 Objetivo
- Aplicar conceitos de **programação em Python**.  
- Utilizar **banco de dados relacional (MySQL)** para persistência de dados.  
- Demonstrar o funcionamento completo de operações **CRUD**.  
- Exercitar boas práticas de organização de código e versionamento com Git/GitHub.  

---

## 🚀 Funcionalidades (CRUD)
- **[Create](ca://s?q=CRUD_Create_em_Python)** → Inserção de novos alunos e disciplinas.  
- **[Read](ca://s?q=CRUD_Read_em_Python)** → Consulta e listagem de registros.  
- **[Update](ca://s?q=CRUD_Update_em_Python)** → Atualização de informações já cadastradas.  
- **[Delete](ca://s?q=CRUD_Delete_em_Python)** → Exclusão de registros do banco.  

---

## 🛠️ Tecnologias Utilizadas

- **[MySQL](ca://s?q=MySQL_banco_de_dados)**
